# Ansible Role: vector

Данная роль предназначена для установки и настройки Vector.

## Требования

Роль была протестирована на следующих операционных системах:
- CentOS/RHEL

## Переменные роли

На данный момент роль не использует переменных.

## Зависимости

Нет.

## Пример использования

Ниже приведен пример того, как использовать эту роль в вашем плейбуке:

    - hosts: servers
      roles:
         - vector-role
